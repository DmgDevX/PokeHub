# PokeHub
Plataforma full-stack desarrollada con Java (Spring Boot) y React (Material UI) que integra múltiples APIs externas (PokéAPI, TCGdex, Open-Meteo y Gemini) para explorar Pokémon y cartas, construir equipos y mazos, simular apariciones según clima y localización, y recibir recomendaciones inteligentes mediante IA.
