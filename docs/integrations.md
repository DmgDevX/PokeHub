# 🌐 Integraciones externas

## PokéAPI
Fuente principal de datos de Pokémon:
- stats
- tipos
- habilidades
- movimientos

## TCGdex
Datos de cartas Pokémon:
- cartas
- sets
- rareza

## Open-Meteo
Datos meteorológicos:
- temperatura
- clima actual
- geolocalización

## Gemini API
Inteligencia artificial para:
- recomendaciones
- análisis de equipos
- generación de estrategias
