# 🤖 Módulo de IA

El módulo de IA utiliza Gemini para generar recomendaciones.

## Funcionalidades

- Generación de equipos
- Creación de mazos
- Análisis estratégico
- Explicaciones detalladas

## Ejemplo de uso

Input:
"Recomiéndame un equipo balanceado contra tipo agua"

Output:
- Lista de Pokémon recomendados
- Explicación de sinergias
- Cobertura de tipos

## Enfoque

No se usa IA de forma aislada:
- Se combina con datos reales del sistema
- Se añade contexto estructurado
